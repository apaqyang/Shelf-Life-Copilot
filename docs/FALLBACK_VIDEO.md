# 2 分钟兜底视频分镜脚本

> 用途：现场断网 / API 故障时秒切的备用素材。**只要总监愿意看完这 2 分钟，demo 算 70% 成功**。
> 配套：[DEMO_SCRIPT.md](DEMO_SCRIPT.md) · [RUNBOOK_DEMO.md](RUNBOOK_DEMO.md) · [demo_samples/](demo_samples/)
> 形式：屏幕录制 + 旁白（不出真人镜头，避免拍摄成本）
> 时长：**严格 ≤ 2:00**，配字幕（客户可能静音看）

---

## 制作准备

| 项 | 建议 |
|---|---|
| 录制工具 | macOS QuickTime / Loom / OBS（任选） |
| 屏幕分辨率 | 1920×1080，字号放大到看片人 1 米外能看清 |
| 输出 | 1080p · MP4 · ≤ 50 MB（企微文件单文件上限考虑） |
| 配音 | 真人录（创始人本人，**不要 TTS** — 客户对真人 + 临期场景的信任感 +30%） |
| 字幕 | **写死烧录**到画面，不依赖播放器加字幕 |
| 客户定制 | A / B 各录一份，把卡片里的厂名 / 品类 / 数字换掉 |

**素材来源**：`docs/demo_samples/customer{A,B}.md` 已是渲染好的 markdown 卡片 — 用 markdown 渲染器（Typora / Obsidian / VSCode 预览）全屏截图，按下面分镜顺序剪。

## 后期与发布
